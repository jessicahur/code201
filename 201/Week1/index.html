An analysis of the JS code on this page

Todays homework assignment is to complete the JS code on this page and then to use other html elments to describe what the code is doing. So I am going to explain the following:

  • The use of the prompt keyword
  • A simple debug output to console
  • The alert keyword

Prompt

var userName = prompt("What is your name?")
The prompt is a useful method for gathering user input. The prompt command will cause a window to pop up for the user, so they can provide input. In this code, The Prompt that the user sees is "What is your name". We are saving the user input entered at the prompt, to the variable userName. By saving the input to a variable, we are going to be able to use the data for providing simple debug statements and to help generate an alert for the user.

Console Output and Debug

console.log("you entered your name as: " + userName);
Leveraging the variable we assigned the prompt value to in the previous paragraph, we can provide a simple debug to console. This is a simplistic, but good enough solution for now. In this case, using the "Console.log" command, we use some basic text string to provide a message. We concatednate the value of the UserName variable to the string, showing what was entered at the prompt by the end user.

The Alert Keyword

alert("It is nice to meet you " + userName + ", I hope you are having a good day");
The last line of code uses the "alert" keyword to provide the end user some feedback. In this case we have two default strings. Between those two strings, you will see that we have the variable name that we declared/defined in the first line of code (Username). We concatenate the two strings and the value of the userName that the end user provided.
